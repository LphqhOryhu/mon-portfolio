Introduction au Projet TimeTrack

Une solution innovante de suivi du temps de travail.

Préparation via des outils de gestion de projet

La phase de préparation a inclus la mise en place de GitHub pour le versioning, Trello pour la gestion des tâches, et Figma pour la conception de l'UI.

Conception de la gestion des données

Le modèle conceptuel de données (MCD) a été établi pour définir la structure des données, accompagné d'un jeu d'essai pour valider le modèle.

Réalisation du projet

Le développement s'est concentré sur la mise en place de procédures stockées pour la gestion des données.

Lexique

Repository : Espace de stockage pour le code source.

Procédures stockées : Ensemble d'instructions SQL pour effectuer des opérations complexes sur les données.

Conclusion

Le projet TimeTrack offre une approche innovante pour le suivi du temps, en apportant précision et efficacité dans la gestion des heures de travail.
